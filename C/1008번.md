> # [A/B](https://www.acmicpc.net/problem/1000)
        
## 1. 문제
   두 수 A, B를 입력 받고, A/B를 출력한다.

## 2. 풀이

1. 변수 두 개를 선언받는다.
2. 입력문으로 변수 두 개를 입력 받는다.
3. 두 변수를  값을 출력한다.

## 3. 소스코드 
```c
#include <stdio.h>

int main() {
	int a, b; scanf("%d %d", &a, &b);
	printf("%.9f\n", (double)a / b);
	return 0;
}
```
